HTML es la base de toda página web. Aprende estos conceptos fundamentales:

## Elementos Esenciales
- Estructura del documento (`<html>`, `<head>`, `<body>`)
- Encabezados y párrafos
- Enlaces y navegación
- Imágenes y multimedia
- Formularios y elementos de entrada

## HTML Semántico
Usar elementos semánticos mejora la accesibilidad y el SEO:
- `<header>`, `<nav>`, `<main>`, `<aside>`, `<footer>`
- `<article>`, `<section>`, `<figure>`
- Jerarquía correcta de encabezados (h1-h6)

## Mejores Prácticas
- Marcado válido y semántico
- Consideraciones de accesibilidad
- Estructura amigable para SEO
- Diseño responsivo mobile-first
