# ¿Qué es Data Science?

La ciencia de datos combina estadística, programación y conocimiento del dominio para extraer insights de datos.

## El proceso de Data Science

```
Pregunta → Datos → Exploración → Modelado → Comunicación
```

1. **Definir el problema**: ¿Qué queremos resolver?
2. **Recopilar datos**: De bases de datos, APIs, web scraping
3. **Limpiar y explorar**: EDA (Exploratory Data Analysis)
4. **Modelar**: Estadística y Machine Learning
5. **Comunicar**: Visualizaciones e informes

## Aplicaciones

| Industria | Aplicación |
|-----------|------------|
| E-commerce | Recomendaciones, predicción de compras |
| Finanzas | Detección de fraude, riesgo crediticio |
| Salud | Diagnóstico, descubrimiento de fármacos |
| Marketing | Segmentación, predicción de churn |
| Transporte | Optimización de rutas, demanda |

## Roles en Data Science

- **Data Analyst**: Análisis y visualización
- **Data Scientist**: Modelos predictivos
- **Data Engineer**: Pipelines y infraestructura
- **ML Engineer**: Poner modelos en producción

## Habilidades necesarias

- Programación (Python, R, SQL)
- Estadística y probabilidad
- Machine Learning
- Visualización de datos
- Comunicación

## Por qué Data Science

- Alta demanda laboral
- Salarios competitivos
- Problemas interesantes
- Impacto real en decisiones
