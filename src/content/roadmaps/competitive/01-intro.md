# ¿Qué es la Programación Competitiva?

La programación competitiva es resolver problemas algorítmicos bajo restricciones de tiempo.

## ¿Por qué practicar CP?

- **Mejores habilidades de resolución de problemas**
- **Entrevistas técnicas**: FAANG usa problemas similares
- **Pensamiento algorítmico**: Optimización y eficiencia
- **Diversión**: Es como resolver puzzles
- **Comunidad global**: Competencias internacionales

## Plataformas Populares

| Plataforma | Descripción |
|------------|-------------|
| [Codeforces](https://codeforces.com) | La más popular, contests semanales |
| [LeetCode](https://leetcode.com) | Ideal para entrevistas |
| [AtCoder](https://atcoder.jp) | Excelentes problemas |
| [CSES](https://cses.fi) | Problem set educativo |
| [HackerRank](https://hackerrank.com) | Bueno para principiantes |

## Competencias Importantes

- **ICPC**: International Collegiate Programming Contest
- **IEEExtreme**: 24 horas de programación
- **Google Code Jam/Kick Start**
- **Meta Hacker Cup**
- **Codeforces Rounds**

## Tu primera competencia

1. Registrarte en Codeforces
2. Participar en Div. 3 o Div. 4
3. No te frustres si resuelves pocos
4. Analiza las soluciones después
5. ¡Practica constantemente!
