# Input/Output Rápido

En competencias, el I/O lento puede causar TLE (Time Limit Exceeded).

## C++ Fast I/O

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    // ¡SIEMPRE incluir estas líneas!
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    
    int n;
    cin >> n;
    
    // Leer array
    vector<int> arr(n);
    for (int i = 0; i < n; i++) {
        cin >> arr[i];
    }
    
    // Leer hasta EOF
    int x;
    while (cin >> x) {
        // procesar x
    }
    
    // Output con newline
    cout << "resultado" << "\n";  // "\n" es más rápido que endl
    
    return 0;
}
```

## Patrones Comunes de Input

```cpp
// Una línea con n números
int n; cin >> n;
vector<int> a(n);
for (int i = 0; i < n; i++) cin >> a[i];

// Matriz n x m
int n, m; cin >> n >> m;
vector<vector<int>> mat(n, vector<int>(m));
for (int i = 0; i < n; i++)
    for (int j = 0; j < m; j++)
        cin >> mat[i][j];

// String
string s; cin >> s;

// Línea completa con espacios
string line;
getline(cin, line);

// T casos de prueba
int t; cin >> t;
while (t--) {
    // resolver caso
}
```

## Python Fast I/O

```python
import sys
input = sys.stdin.readline

n = int(input())
arr = list(map(int, input().split()))
```

## Tips

- Usa `"\n"` en lugar de `endl`
- Evita `printf/scanf` mezclado con `cin/cout`
- Para output masivo, acumula en string
