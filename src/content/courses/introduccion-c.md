# Introducción a C

Este curso está diseñado para estudiantes con conocimientos básicos de programación que desean aprender C, un lenguaje poderoso y versátil utilizado en programación competitiva, desarrollo de sistemas, videojuegos y aplicaciones de alto rendimiento. Durante 2 semanas, los estudiantes adquirirán una sólida comprensión de los principios de C y desarrollarán habilidades prácticas para resolver problemas utilizando el lenguaje.

## Objetivos del curso

Al finalizar este curso, los estudiantes serán capaces de:

- Comprender la sintaxis y características específicas de C.
- Implementar programas utilizando estructuras de datos fundamentales.
- Aplicar el paradigma de programación estructurada en C.
- Desarrollar soluciones para problemas de programación.
- Comprender las características de C moderno (C17).

## Metodología

El curso combina:
- Clases teórico-prácticas semanales (2 horas, 3 días a la semana)
- Ejercicios prácticos para realizar en casa
- Resolución de problemas
- Proyecto final integrador

## Contenido del curso

### Clase 1 - Introducción y sintaxis básica
- Presentación del curso y objetivos
- Sintaxis básica y semántica
- Primer programa: compilación y ejecución
- Comentarios y directivas del preprocesador

### Clase 2 - Tipos, variables, operadores y E/S
- Variables y tipos primitivos
- Operadores y expresiones
- Instrucciones básicas de entrada y salida de datos (printf/scanf o iostream)
- Práctica guiada de I/O

### Clase 3 - Control de flujo y funciones
- Estructuras condicionales (if, else, switch)
- Estructuras repetitivas (for, while, do-while)
- Funciones: definición, parámetros, retorno y ámbito
- Paso de parámetros por valor y por referencia

### Clase 4 - Arrays y estructuras
- Arreglos (arrays) y acceso a datos
- Estructuras (registros) y representación en memoria
- Manipulación y ejemplos prácticos combinando arrays y structs

### Clase 5 - Punteros y memoria dinámica
- Concepto de puntero y operadores & y *
- Aritmética de punteros y arreglos de punteros
- Paso de direcciones a funciones
- Gestión dinámica básica (malloc/free o new/delete) y consideraciones de seguridad

### Clase 6 - Archivos, integración y proyecto final
- Archivos: apertura, lectura, escritura y cierre
- Formatos simples y parsing básico
- Repaso integrador de las dos unidades
- Proyecto final: especificación, criterios de evaluación y siguientes pasos
