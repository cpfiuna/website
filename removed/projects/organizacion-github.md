## Nuestra organización en GitHub

Creamos la organización de GitHub del Club de Programación FIUNA para centralizar el trabajo, los proyectos y los experimentos de la comunidad en un solo lugar. La organización está verificada con nuestro dominio oficial `cpfiuna.io`, lo que nos permite mostrar identidad institucional y gestionar permisos y repositorios con mayor seguridad.

Dentro de la organización alojamos repositorios de distintos tipos:

- Proyectos productivos (sites, herramientas internas)
- Repositorios educativos y de laboratorio (experimentos, prototipos)
- Recursos y guías para contribuir (templates, plantillas de PR y issues)

Cada repositorio actúa como un pequeño espacio de aprendizaje: los integrantes pueden probar tecnologías nuevas, documentar resultados y practicar flujos reales de desarrollo sin riesgo para servicios críticos.

## `mi-primer-pr`: una puerta de entrada para contribuir

Uno de los repositorios destacados dentro de la organización es `mi-primer-pr`. Está diseñado para que quienes nunca usaron Git o GitHub puedan:

- Aprender el flujo básico: fork → clone → branch → commit → PR
- Practicar comandos esenciales de Git en un entorno seguro
- Hacer su primera contribución real con revisión por pares

La actividad central del repo es muy simple y simbólica: hacer un commit de una sola línea,una frase corta, una pequeña dedicatoria o un mensaje divertido, que quedará en la historia del proyecto como un lindo easter egg para futuras generaciones del club.

Cómo participar en `mi-primer-pr` (resumen rápido):

- Forkeá `https://github.com/cpfiuna/mi-primer-pr`.
- Cloná tu fork: `git clone <url-del-fork>`.
- Creá una rama nueva: `git checkout -b mi-primer-pr-<tu-nombre>`.
- Editá el archivo indicado o agregá una línea nueva.
- Hacé un commit con una sola línea descriptiva: `git commit -m "Mi línea: presente en mi-primer-pr"`.
- Hacé push y abrí un Pull Request hacia `cpfiuna/mi-primer-pr`.

Ese pequeño gesto queda registrado en el historial y es un recurso pedagógico: enseña buenas prácticas (buen mensaje de commit, ramas, PR) y al mismo tiempo crea un recuerdo colectivo.

Invitamos a todos los miembros del Club de Programación FIUNA a sumarse a la organización de GitHub y a contribuir en `mi-primer-pr`. Es una excelente oportunidad para aprender, colaborar y dejar una marca personal en la historia del club. ¡Esperamos ver muchas contribuciones pronto!
