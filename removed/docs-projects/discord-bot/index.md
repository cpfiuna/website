# Documentación del bot — índice

Este directorio contiene la documentación principal del CPF Discord Bot. A continuación tienes los archivos clave y una breve descripción para facilitar la navegación:

- `index.md` — Índice (este archivo).
- `getting-started.md` — Cómo correr el bot localmente para desarrollo y pruebas.
- `deployment.md` — Guía detallada para desplegar y actualizar el bot en una VM (ej.: Azure B2) usando `pm2`.
- `architecture.md` — Estructura y componentes del proyecto.
- `api.md` — Información sobre registro de comandos y estructura de comandos slash.
- `contributing.md` — Pautas para contribuir al proyecto (estilo de commits, branch, pruebas).

Usá estos documentos como guía principal para operar, contribuir y mantener el bot.

Si necesitás ayuda específica para la VM o credenciales, contactá a los administradores del proyecto.
